PROGRAMMING CLUB SUMMER PROJECT

Work Done So Far

  • Till last evaluation our front and back end were on two different apps. Now, we have integrated our front and back end in a single app.
  • While working with the app, we found that the app didn't work if the phone goes to sleep mode. So, to make the app run even in sleep mode we tried implementing wakelock and wakefulbroadcast for the task and found wakelock to be more simpler and useful. And now the app is working even when the phone is in sleep mode.
  • We have successfully implemented the missed call feature of the app which allows the app to send the missed calls to another phone when requested via a passcode. And since large messages cannot be sent in a single message, we implemented multi parts message to complete the task.
  • We tried Shared Preferences and Database to store the input such as passcode that the user provides for differnt applications and finally implemented it using Shared preferences as it was much easier and relevant than database.
  • The app now allows user to enable or disable different options.
  • We have also successfully implemented the IMEI, alarm ringer and profile changer features in the app.
  • We are also adding a feature to the app which will show all the actions taken by the app in a list and are almost done with it.
  • The app also shows a notification whenever the app responds to a message. As soon as the list is integrated into the app, we will connect the notification to it.
  • We have also worked on detecting SIM change feature which will send a message to a predefined number about the new phone number, whenever the SIM is changed but are having some difficulties in implementing it.
